# 6S Audit - Frontend

## Overview
6S Audit is a digital system designed to monitor and improve workplace organization and efficiency based on the 6S principles: Sort, Set in order, Shine, Standardize, Sustain, and Safety. This repository contains the frontend implementation built using React Vite and Mantine UI, tailored for PT MPI's LEAN manufacturing practices.

## Features
- Audit Schedule Management
- Digital 6S Audit Form
- Audit History Tracking
- Auditor & Area Selection
- Scoring & Comment System

Responsive and User-Friendly Interface

## Tech Stack
- React.js + Vite
- Mantine UI Framework
- React Router DOM
- Tabler Icons
- Axios

## Prerequisites
- Node.js (version 16 or higher)
- npm or yarn package manager

## Installation

1. Clone the repository
```bash
git clone https://github.com/zmafnan/6s-frontend.git
cd bts-frontend
```

2. Install dependencies
```bash
npm install
# or
yarn install
```
3. Start the development server
```bash
npm run dev
# or
yarn dev
```
